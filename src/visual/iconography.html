Icon Characteristics

  • Consistent line weight
  • Simple, recognizable forms
  • Aligned with brand aesthetic
  • Scalable and clear at all sizes
## Icon Sizes Standard icon sizes for different applications: - **Small**: 16px - For inline use and small UI elements - **Medium**: 24px - Standard size for most applications - **Large**: 32px - For emphasis and larger UI elements - **Extra Large**: 48px+ - For hero sections and feature highlights ## Usage Guidelines 1. Use icons consistently across all applications 2. Maintain consistent style and weight 3. Ensure icons are recognizable and clear 4. Use appropriate sizes for context 5. Provide text labels when meaning isn't immediately clear ## Icon Library Our icon library includes: - Navigation icons - Action icons - Status icons - Content type icons - Social media icons ## Best Practices - Use icons to enhance, not replace, text when possible - Maintain consistent spacing around icons - Ensure sufficient contrast for visibility - Use icons that align with brand style - Test icons at various sizes for clarity
