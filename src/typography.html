--- layout: layouts/base.njk title: Typography description: Typography guidelines and text styles breadcrumbs: - label: Home url: / - label: Typography url: /typography category: Design System --- ## Typography System Our typography system provides a consistent hierarchy and readable text across all platforms. ## Font Families We use system fonts for optimal performance and cross-platform consistency: ```css font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, sans-serif; ``` ## Type Scale Our type scale follows a harmonious progression:

Heading 1 - 2.5rem / 40px

Heading 2 - 1.875rem / 30px

Heading 3 - 1.5rem / 24px

Heading 4 - 1.25rem / 20px

Body Text - 1rem / 16px

Small Text - 0.875rem / 14px

## Text Styles ### Headings Headings provide structure and hierarchy to your content: ```html

Main Page Title

Section Heading

Subsection Heading

Minor Heading

``` ### Body Text Body text should be readable and comfortable: ```html

This is body text. It should be easy to read and provide good contrast against the background.

``` ### Emphasis Use emphasis sparingly to highlight important information: ```html

This is bold text and this is italic text.

``` ## Line Height We use a line height of 1.6 for body text to ensure readability: ```css line-height: 1.6; ``` ## Color Text colors follow our color system: - **Primary text**: `var(--wa-color-neutral-10)` - **Secondary text**: `var(--wa-color-neutral-30)` - **Muted text**: `var(--wa-color-neutral-40)` ## Best Practices 1. Use headings in order (h1 → h2 → h3) to maintain hierarchy 2. Keep paragraphs concise and scannable 3. Use bold and italic sparingly for emphasis 4. Ensure sufficient contrast for accessibility 5. Maintain consistent spacing between elements
