### Available Styles - **Regular** - Standard upright style - **Italic** - Slanted style for emphasis - **Bold** - Heavy weight for strong emphasis - **Bold Italic** - Combination of bold and italic ### Usage Guidelines - Use lighter weights for large display text - Use regular weight for body text - Use medium to semi-bold for subheadings - Use bold sparingly for strong emphasis - Avoid using more than 2-3 weights in a single design ## Usage Guidelines Proper application of typography ensures clear communication and maintains brand consistency. ### Type Scale Our type scale provides consistent sizing across all applications:

Heading 1 - 2.5rem / 40px

Heading 2 - 1.875rem / 30px

Heading 3 - 1.5rem / 24px

Heading 4 - 1.25rem / 20px

Body Text - 1rem / 16px

Small Text - 0.875rem / 14px

### Line Height - **Headlines**: 1.2 - 1.3 (tighter for impact) - **Body Text**: 1.5 - 1.6 (comfortable reading) - **Small Text**: 1.4 - 1.5 (maintains readability) ### Letter Spacing - **Headlines**: Slightly tighter for cohesion - **Body Text**: Default spacing - **All Caps**: Slightly increased for readability ### Best Practices 1. **Hierarchy**: Use size and weight to create clear hierarchy 2. **Contrast**: Ensure sufficient contrast between text and background 3. **Alignment**: Use left alignment for body text, center for headlines sparingly 4. **Line Length**: Keep body text to 45-75 characters per line 5. **Spacing**: Use consistent spacing between elements ### Common Mistakes to Avoid - Using too many different font sizes - Insufficient line height for readability - Poor contrast between text and background - Inconsistent spacing - Mixing too many weights in one design
