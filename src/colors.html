## Semantic Colors Our design system includes semantic colors for different states and messages: - **Success**: Used for positive actions and success messages - **Warning**: Used for warnings and cautionary messages - **Danger**: Used for errors and destructive actions - **Info**: Used for informational messages ## Usage Guidelines 1. **Primary colors** should be used sparingly for emphasis and calls to action 2. **Neutral colors** form the foundation of most designs 3. **Semantic colors** should be used consistently for their intended purposes 4. Always ensure sufficient contrast for accessibility ## CSS Variables All colors are available as CSS variables: ```css /* Primary colors */ --wa-color-primary-50 --wa-color-primary-500 --wa-color-primary-900 /* Neutral colors */ --wa-color-neutral-0 --wa-color-neutral-500 --wa-color-neutral-900 ```
