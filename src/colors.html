## Semantic Colors Our design system includes semantic colors for different states and messages: - **Success**: Used for positive actions and success messages - **Warning**: Used for warnings and cautionary messages - **Danger**: Used for errors and destructive actions - **Info**: Used for informational messages ## Usage Guidelines 1. **Primary colors** should be used sparingly for emphasis and calls to action 2. **Neutral colors** form the foundation of most designs 3. **Semantic colors** should be used consistently for their intended purposes 4. Always ensure sufficient contrast for accessibility ## CSS Variables All colors are available as CSS variables: ```css /* Brand colors */ --wa-color-brand-95 --wa-color-brand-50 --wa-color-brand-10 /* Neutral colors */ --wa-color-neutral-95 --wa-color-neutral-50 --wa-color-neutral-10 ```
