
## Image Specifications

### Logo Images
- **Format**: PNG with transparency (for logos), SVG preferred for scalable logos
- **Resolution**: Minimum 300 DPI for print, optimized for web
- **Background**: Transparent or white background as appropriate

### Color Swatches
- **Format**: PNG or JPG
- **Resolution**: High resolution for accurate color representation
- **Size**: Standard swatch sizes (e.g., 200x200px minimum)

### Typography Specimens
- **Format**: PNG
- **Resolution**: High resolution for readability
- **Content**: Full character sets, weights, and styles

### Photography
- **Format**: JPG
- **Resolution**: High resolution, optimized for web
- **Style**: Consistent with brand photography guidelines

### Application Examples
- **Format**: PNG or JPG
- **Resolution**: High resolution for clarity
- **Content**: Mockups or actual examples of branded materials

## Notes

- All images should be optimized for web use while maintaining quality
- Use appropriate file formats (PNG for graphics with transparency, JPG for photos)
- Ensure all images follow brand guidelines for color, style, and quality
- Update this README when adding new images or categories
